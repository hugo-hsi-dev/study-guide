---
title: keys()
description: Creating an iterator for array indices
---

## Basics

- Creates a new iterator object containing keys
- Returns that iterator object

## Usage

```js
const array1 = ['a', 'b', 'c'];
const iterator = array1.keys();

for (const key of iterator) {
  console.log(key);
}

// Expected output: 0
// Expected output: 1
// Expected output: 2
```
