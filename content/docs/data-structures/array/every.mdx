---
title: every()
description: test every value in an array
---

## Basics

- This method tests each element in an array
- Returns true if every iteration returns true
- If any given iteration returns false, the loop will stop and method will return false

## Usage

```js
const isBelowThreshold = (currentValue) => currentValue < 40;

const array1 = [1, 30, 39, 29, 10, 13];

console.log(array1.every(isBelowThreshold));
// Expected output: true
```
