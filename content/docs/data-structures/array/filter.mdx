---
title: filter()
description: Filters out elements from an array
---

## Basics

- Takes in a callback function
- Function should return true if should be kept in array
- Function should return false if it should be removed

## Usage

```js
const words = ['spray', 'elite', 'exuberant', 'destruction', 'present'];

const result = words.filter((word) => word.length > 6);

console.log(result);
// Expected output: Array ["exuberant", "destruction", "present"]
```
