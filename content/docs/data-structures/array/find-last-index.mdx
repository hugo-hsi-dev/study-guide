---
title: findLastIndex()
description: Reverse search find index
---

## Basics

- Finds the index of a given element
- Takes a cb function which will return a boolean
- True means give us the index of the element
- False means continue searching

## Usage

```js
const array1 = [5, 12, 50, 130, 44];

const isLargeNumber = (element) => element > 45;

console.log(array1.findLastIndex(isLargeNumber));
// Expected output: 3
// Index of element with value: 130
```
