---
title: reverse()
description: Reverses the order of an array
---

## Basics

- Reverses the array in place
- Returns a reference to the now reversed array

## Usage

```js
const array1 = ['one', 'two', 'three'];
console.log('array1:', array1);
// Expected output: "array1:" Array ["one", "two", "three"]

const reversed = array1.reverse();
console.log('reversed:', reversed);
// Expected output: "reversed:" Array ["three", "two", "one"]

// Careful: reverse is destructive -- it changes the original array.
console.log('array1:', array1);
// Expected output: "array1:" Array ["three", "two", "one"]
```

<Callout
  type='info'
  title='Return Value'
>
  The return value is only a reference, no copy is made. If you wanted to
  reverse the reference, the original one will be modified too
</Callout>
