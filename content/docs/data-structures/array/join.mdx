---
title: join()
description: Joins an array into a string
---

## Basics

- Takes an optional argument of what to put between elements in the resulting string
- If no value is given, will separate elements with a ',' without a space

## Usage

```js
const elements = ['Fire', 'Air', 'Water'];

console.log(elements.join());
// Expected output: "Fire,Air,Water"

console.log(elements.join(''));
// Expected output: "FireAirWater"

console.log(elements.join('-'));
// Expected output: "Fire-Air-Water"
```

## Quirks

### Nested Arrays

- If there are nested arrays, those arrays will also be parsed into strings
- The toString() method on strings remove the wrapping brackets, making it look like the arrays were flattened, while the reason may be different
- The separator of nested arrays cannot be modified

```js
const matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9],
];

console.log(matrix.join()); // 1,2,3,4,5,6,7,8,9
console.log(matrix.join(';')); // 1,2,3;4,5,6;7,8,9
```

### Undefined values and sparse arrays

- If there are any undefined values, there will still be separators between real values and undefined values

```js
console.log([1, , 3].join()); // '1,,3'
console.log([1, undefined, 3].join()); // '1,,3'
```
